# 🔍 Clue Notes - Detective Digital

**Clue Notes** es una aplicación de asistencia para el juego de mesa **Clue** (o Cluedo). Diseñada para reemplazar la libreta de papel tradicional, esta herramienta te permite llevar un registro impecable de las pistas, sospechosos, armas y lugares de forma rápida, privada y eficiente.

¡Optimiza tus deducciones y conviértete en el mejor detective de la partida!

## ✨ Características Principales

* **Gestión Dinámica:** Soporte para partidas de **3 a 6 jugadores**.
* **Interfaz Inteligente:** Registro de cartas propias (Mano), descartes de oponentes y confirmaciones del sobre.
* **Algoritmo de Probabilidad:** El sistema marca automáticamente con un rayo (⚡) las cartas con alta probabilidad de ser el veredicto final.
* **Veredicto Rápido:** Acceso directo a una vista resumida de tus deducciones finales mediante el botón del Mazo de Juez.
* **Diseño Inmersivo:** Tema oscuro (Dark Mode) con colores personalizados según el personaje que elijas (Sr. Verduzco, Srta. Escarlata, etc.).
* **Siempre Encendido:** La pantalla se mantiene activa durante la partida para que no pierdas tiempo desbloqueando tu teléfono.

## 🛠️ Tecnologías Usadas

* **Lenguaje:** Kotlin
* **Framework:** Jetpack Compose (UI Moderna y Declarativa)
* **Arquitectura:** State Management con `mutableStateOf` y `derivedStateOf`.
* **Android:** API 24+ (Android 7.0 en adelante).
